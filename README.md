# key_function_replace

<h3>Descrption</h3>
<p>A few keyboard buttons where not working on my dad's laptop (5, 6, -, +), so I created caps lock like alternative to
    type those keys</p>

<h3> Instructions</h3>
press shift+caps_lock to start (and stop) <br> then
<ul>
    <li>
        press 1 for 5
    </li>
    <li>
        press 2 for 6</li>

    <li>
        press 3 for %</li>

    <li>
        press 4 for ^</li>

    <li>
        press 7 for -</li>

    <li>
        press 8 for +</li>
    <li>
        press 9 for _
    </li>
    <li>
        press 0 for =
    </li>
</ul>
